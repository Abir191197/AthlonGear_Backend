## API Reference

#### Product Create

```http
  POST api/products/createProduct
```
#### Get All Product

```http
  GET api/products
```

#### Get Single  Product

```http
  GET api/products/:{id}
```
#### DELETE OR UPDATE Product

```http
  PUT api/products/:{id}
```
#### Create Order

```http
  POST api/orders/createOrder
```
#### Get Single Order

```http
  GET api/orders/:{orderId}
```
